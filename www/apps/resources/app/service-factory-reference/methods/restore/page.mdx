## Resore One Record

```ts
const restoredPosts = await customModuleService.restorePosts("123")
```

To restore one record, pass its ID as a parameter of the method.

The method returns an object, whose keys are of the format `{camel_case_data_model_name}_id` (for example, `post_id`), and their values are arrays of restored records' IDs.

---

## Restore Multiple Records

```ts
const restoredPosts = await customModuleService.restorePosts([
  "123",
  "321"
])
```

To restore multiple records, pass an array of IDs as a parameter of the method.

The method returns an object, whose keys are of the format `{camel_case_data_model_name}_id` (for example, `post_id`), and their values are arrays of restored records' IDs.

---

## Restore Records Matching Filters

```ts
const restoredPosts = await customModuleService.restorePosts({
```
