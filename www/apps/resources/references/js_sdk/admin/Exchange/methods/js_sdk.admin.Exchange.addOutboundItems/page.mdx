# addOutboundItems

This method adds outbound (or new) items to an exchange. 
These outbound items will have the action `ITEM_ADD`.

This method sends a request to the [Add Outbound Items](https://docs.medusajs.com/v2/api/admin#exchanges\_postexchangesidoutbounditems)
API route.

## Example

```ts
sdk.admin.exchange.addOutboundItems("exchange_123", {
  items: [{
    id: "variant_123",
    quantity: 1
  }]
})
.then(({ exchange }) => {
  console.log(exchange)
})
```
