# retrieve

This method retrieves an exchange by its ID. It sends a request to the 
[Get Exchange](https://docs.medusajs.com/v2/api/admin#exchanges\_getexchangesid)
API route.

## Example

To retrieve an exchange by its ID:

```ts
sdk.admin.exchange.retrieve("exchange_123")
.then(({ exchange }) => {
  console.log(exchange)
})
```

To specify the fields and relations to retrieve:

```ts
sdk.admin.exchange.retrieve("exchange_123", {
  fields: "id,*order"
})
.then(({ exchange }) => {
  console.log(exchange)
})
```

Learn more about the `fields` property in the [API reference](https://docs.medusajs.com/v2/api/store#select-fields-and-relations).
