# addOutboundShipping

This method adds an outbound shipping method to an exchange. The outbound shipping method 
will have a `SHIPPING_ADD` action.

This method sends a request to the [Add Outbound Shipping](https://docs.medusajs.com/v2/api/admin#exchanges\_postexchangesidoutboundshippingmethod)
API route.

## Example

```ts
sdk.admin.exchange.addOutboundShipping("exchange_123", {
  shipping_option_id: "so_123"
})
.then(({ exchange }) => {
  console.log(exchange)
})
```
