# create

This method creates an admin exchange. It sends a request to the
[Create Exchange](https://docs.medusajs.com/v2/api/admin#exchanges\_postexchanges) API route.

## Example

```ts
sdk.admin.exchange.create({
  order_id: "order_123"
})
.then(({ exchange }) => {
  console.log(exchange)
})
```
