# update

This method updates an API key's details. It sends a request to the 
[Update API Key](https://docs.medusajs.com/v2/api/admin#api-keys\_postapikeysid) API route.

## Example

```ts
sdk.admin.apiKey.update("apk_123", {
  title: "Development"
})
.then(({ api_key }) => {
  console.log(api_key)
})
```
