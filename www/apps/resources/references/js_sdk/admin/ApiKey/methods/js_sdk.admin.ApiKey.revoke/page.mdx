# revoke

This method revokes an API key. It sends a request to the 
[Revoke API Key](https://docs.medusajs.com/v2/api/admin#api-keys\_postapikeysidrevoke) API route.

## Example

```ts
sdk.admin.apiKey.revoke("apk_123")
.then(({ api_key }) => {
  console.log(api_key)
})
```
