# create

This method creates a fulfillment. It sends a request to the 
[Create Fulfillment](https://docs.medusajs.com/v2/api/admin#fulfillments\_postfulfillments)
API route.

## Example

```ts
sdk.admin.fulfillment.create({
  location_id: "sloc_123",
  provider_id: "my_fulfillment",
  delivery_address: {
    country_code: "us"
  },
  items: [
    {
      title: "Shirt",
      sku: "SHIRT",
      quantity: 1,
      barcode: "123"
    }
  ],
  labels: [],
  order: {},
  order_id: "order_123"
})
.then(({ fulfillment }) => {
  console.log(fulfillment)
})
```
