# cancel

This method cancels a fulfillment. It sends a request to the
[Cancel Fulfillment](https://docs.medusajs.com/v2/api/admin#fulfillments\_postfulfillmentsidcancel)
API route.

## Example

```ts
sdk.admin.fulfillment.cancel("ful_123")
.then(({ fulfillment }) => {
  console.log(fulfillment)
})
```
