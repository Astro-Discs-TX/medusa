# createShipment

This method creates a shipment for a fulfillment. It sends a request to the
[Create Shipment](https://docs.medusajs.com/v2/api/admin#fulfillments\_postfulfillmentsidshipment)
API route.

## Example

```ts
sdk.admin.fulfillment.createShipment("ful_123", {
  labels: [
    {
      tracking_number: "123",
      tracking_url: "example.com",
      label_url: "example.com"
    }
  ]
})
.then(({ fulfillment }) => {
  console.log(fulfillment)
})
```
