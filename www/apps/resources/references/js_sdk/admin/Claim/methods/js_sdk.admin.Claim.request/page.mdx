# request

This method confirms a claim request, applying its changes on the associated order.
It sends a request to the [Confirm Claim Request](https://docs.medusajs.com/v2/api/admin#claims\_postclaimsidrequest)
API route.

## Example

```ts
sdk.admin.claim.request(
  "claim_123", 
  {},
)
.then(({ claim }) => {
  console.log(claim)
})
```
