# PriceSetMoneyAmountRulesDTO

An object representing a price set money amount rule, which holds data related to the association between a price set money amount and a rule.

## Properties

### id

 **id**: `string`

A string indicating the ID of the price set money amount.

___

### price\_set\_money\_amount

 **price\_set\_money\_amount**: [`PriceSetMoneyAmountDTO`](PriceSetMoneyAmountDTO.mdx)

an object of type [PriceSetMoneyAmountDTO](PriceSetMoneyAmountDTO.mdx) holding the data of the associated price set money amount.

___

### rule\_type

 **rule\_type**: [`RuleTypeDTO`](RuleTypeDTO.mdx)

an object of type [RuleTypeDTO](RuleTypeDTO.mdx) holding the data of the associated rule type.

___

### value

 **value**: `string`

a string indicating the value of the price set money amount rule.
