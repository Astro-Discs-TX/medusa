# AddPricesDTO

An object used to specify prices to add to a price set.

## Properties

### priceSetId

 **priceSetId**: `string`

A string indicating the ID of the price set to add prices to.

___

### prices

 **prices**: [`CreatePricesDTO`](CreatePricesDTO.mdx)[]

An array of objects of type [CreatePricesDTO](CreatePricesDTO.mdx), each being a price to add to the price set.
