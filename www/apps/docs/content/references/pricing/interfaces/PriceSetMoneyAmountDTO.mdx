# PriceSetMoneyAmountDTO

An object representing a price set money amount, which holds the data related to the association between a price set and a money amount.

## Properties

### id

 **id**: `string`

a string indicating the ID of a price set money amount.

___

### money\_amount

 `Optional` **money\_amount**: [`MoneyAmountDTO`](MoneyAmountDTO.mdx)

an object of type [MoneyAmountDTO](MoneyAmountDTO.mdx) holding the data of the associated money amount.

___

### price\_set

 `Optional` **price\_set**: [`PriceSetDTO`](PriceSetDTO.mdx)

an object of type [PriceSetDTO](PriceSetDTO.mdx) holding the data of the associated price set.

___

### title

 `Optional` **title**: `string`

a string indicating the title of the price set money amount.
