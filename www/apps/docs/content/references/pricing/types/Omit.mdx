# Omit

 **Omit**<`T`, `K`\>: [`Pick`](Pick.mdx)<`T`, [`Exclude`](Exclude.mdx)<keyof `T`, `K`\>\>

Construct a type with the properties of T except for those in type K.

#### Type parameters

| Name | Type |
| :------ | :------ |
| `T` | `object` |
| `K` | keyof `any` |
