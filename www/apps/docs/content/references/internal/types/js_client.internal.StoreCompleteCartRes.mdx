# StoreCompleteCartRes

 **StoreCompleteCartRes**: ``{ data: [Cart](../classes/js_client.internal.Cart.mdx) ; type: "cart"  }`` \| ``{ data: [Order](../classes/js_client.internal.Order.mdx) ; type: "order"  }`` \| ``{ data: [Swap](../classes/js_client.internal.Swap.mdx) ; type: "swap"  }``
